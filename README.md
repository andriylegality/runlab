# RUNLAB — Personalized Race Intelligence

> Built by **andriylegality** · Balikpapan, Indonesia 🇮🇩

A mobile-first web app for personalized running pacing & hydration management. Built for the running community — especially for events like **Sahur Run Dalkot Balikpapan**.

---

## Features

- **Pacing Management** — personalized per segment based on actual training data, age, gender, and VO2Max
- **HR Zone Calculator** — Heart Rate Reserve method (most accurate)
- **Run-Walk Protocol** — auto-adjusted from weekly mileage
- **Hydration Management** — schedule based on available nutrition (salt caps, gel, isotonic, dates)
- **Race Plan Ticket** — summary with estimated finish time
- **Bilingual** — Bahasa Indonesia & English
- **Offline-ready** — data saved in localStorage

---

## Tech Stack

- React 18 + Vite
- Pure CSS-in-JS (no UI libraries)
- localStorage for data persistence
- HYROX-inspired design aesthetic
